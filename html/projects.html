A file transfer protocol is a system that handles the sending of files from computer
to computer via a server and client - a project commonly tackled by beginners to the
field. This is my version of a FTP - it stands unfinished, however I plan to fully
finish it one of these days. Maybe two...

Not sure how long it took me to make, I'd say about a week. It really taught me alot
about the server/client relationship in a TCP/UDP sense. I'd relaly suggest anyone looking
to learn to code make this their first larger-scale project.

Check it out on Github!
